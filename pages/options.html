Default filtering mode

Atur mode pemfilteran default untuk website. Kamu bisa ganti cepat per kebutuhan: off, normal, atau optimal.

Off

Semua pemblokiran dimatikan. Cocok untuk troubleshooting atau whitelist cepat.

Normal

Blokir jaringan iklan dari blocklist, tanpa filter elemen HTML tambahan.

Optimal

Proteksi penuh: blokir jaringan dan blokir elemen iklan langsung di halaman.

About

Coffey membantu memblokir request iklan dan menyembunyikan elemen iklan yang mengganggu. Mode dapat diubah kapan saja dari popup atau halaman settings ini.
