Panduan Instalasi Manual

Pasang Coffey
Dalam 5 Menit

1
Download repositori: klik Code → Download ZIP di GitHub, atau jalankan git clone https://github.com/Bruhrbx/Coffey.git
2
Ekstrak file ZIP ke folder khusus di komputermu.
3
Buka browser dan kunjungi chrome://extensions/
4
Aktifkan Developer Mode di pojok kanan atas.
5
Klik Load Unpacked → pilih folder Coffey yang berisi manifest.json
6
Selesai! ☕ Coffey kini aktif dan memblokir iklan di browsermu.
1
Download dan ekstrak repositori Coffey dari GitHub.
2
Buka Firefox dan ketik about:debugging di address bar, lalu Enter.
3
Klik This Firefox di menu kiri.
4
Klik Load Temporary Add-on... → pilih file manifest.json
5
Coffey aktif! Untuk instalasi permanen, gunakan Firefox Developer Edition atau Nightly.
⚠️ Coffey masih dalam fase Beta / Unstable. Mungkin ada bug minor. Laporkan masalah via GitHub Issues.
